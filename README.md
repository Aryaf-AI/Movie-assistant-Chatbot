# Movie-assistant-Chatbot
AI chatbot that provides a personalized movie recommendations.
🎬 AI Movie Assistant Chatbot
📝 Overview
An intelligent conversational agent designed to help users discover, explore, and learn about movies. Built using Python, this AI assistant provides personalized movie recommendations, trivia, and information about actors, directors, and cinematic trends.

✨ Features
Smart Recommendations: Suggests movies based on user preferences, genres, or moods.

Cast & Crew Insights: Retrieves details about specific actors, directors, and character roles.

Box Office & Trivia: Can answer general queries about movie rankings, historical milestones, and pop culture.

💬 Example Queries
Here are a few things you can ask the assistant:

"Can you recommend a Comedy movie ?"

"What movies has Cillian Murphy starred in?"

"tell me about inception"
